Introducing smarter building blocks to create more consistent customer experiences at scale.

Copy

You may notice build warnings if your consuming application is using Node < 20. These can be safely ignored for consumption.

If your project depends on external modules that also use Cedar, update those to the same major version and keep Cedar as a peer dependency.

## v15.5.0

Copy

### Object overlay: Flexible content positioning

The new `CdrObjectOverlay` component provides flexible positional overlays relative to a container.

Key features:

- Flexible positioning in 9 locations
- Responsive position, margin, and gradient options
- Gradient support for readability
- Spacing control for margin and padding

Learn more: [Object Overlay](https://cedar.rei.com/components/object-overlay).

### Surface navigation: Enhanced interactive containers

`CdrSurfaceNavigation` adds interaction-state patterns for navigational containers.

Key features:

- State management for rest, hover, active, focused, and disabled
- Flexible building block for cards and navigation tiles

Learn more: [Surface Navigation](https://cedar.rei.com/components/surface#navigation).

## v15.1.0

Copy

### Fluid space: Enhancing layout flexibility

Fluid space uses `clamp()` and container query units to scale spacing with screen and container size.

```css
--cdr-space-scale-0: clamp(0.2rem, 0.2rem + 0.11cqi, 0.3rem);
--cdr-space-scale-1: clamp(0.3rem, 0.3rem + 0.11cqi, 0.4rem);
--cdr-space-scale-2: clamp(0.4rem, 0.4rem + 0.11cqi, 0.5rem);
--cdr-space-scale-3: clamp(0.8rem, 0.7rem + 0.22cqi, 1rem);
--cdr-space-scale-4: clamp(1.2rem, 1.1rem + 0.33cqi, 1.5rem);
--cdr-space-scale-5: clamp(1.6rem, 1.5rem + 0.44cqi, 2rem);
--cdr-space-scale-6: clamp(2.4rem, 2.2rem + 0.66cqi, 3rem);
--cdr-space-scale-7: clamp(3.2rem, 2.9rem + 0.88cqi, 4rem);
--cdr-space-scale-8: clamp(4.8rem, 4.4rem + 1.31cqi, 6rem);

/* One-up pairs */
--cdr-space-scale-0--1: clamp(0.2rem, 0.1rem + 0.223cqi, 0.4rem);
--cdr-space-scale-3--4: clamp(0.4rem, 0.2rem + 0.66cqi, 1rem);
--cdr-space-scale-3--5: clamp(0.8rem, 0.1404rem + 1.21cqi, 1.6rem);
```

## Breaking changes

Copy

### CdrImg refactor

Removals:

- Removed `cover` prop, use `fit`
- Removed `crop` prop, use `position`
- Removed `coverClass` prop
- Removed `modifier` prop

Changes:

- Updated `radius` prop to support Cedar token shorthand or CSS value
- Updated `ratio` to map directly to `aspect-ratio`

Additions:

- Added `position`
- Added `fit`
- Added `srcset` and `sizes`
