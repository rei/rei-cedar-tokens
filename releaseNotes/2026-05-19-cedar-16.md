# Cedar 16

Filmstrip and Surface additions, plus a refactored Sass codebase.

## Update steps

Copy

| package name                 | version |
| ---------------------------- | ------- |
| @rei/cedar                   | ^16.x.x |
| @rei/cdr-tokens              | ^13.x.x |
| @rei/cdr-component-variables | ^10.x.x |

If your project depends on external modules that also use Cedar, update those modules to the same major version and keep Cedar as a peer dependency.

### v16.0.2

Fixes fluid custom properties after Sass module migration so fluid variables are available in built CSS artifacts.

## New features

Copy

### Adding CdrSurfaceScroll

`CdrSurfaceScroll` extends Surface with controlled overflow and custom scrollbar handling for scrollable content.

### Introducing CdrFilmstrip

Filmstrip is a supported pattern using SurfaceScroll for accessible, scrollable media and content tracks.

Key features:

- Full keyboard focus parity
- New `scrollNavigate` event for interaction handling

### Public injection keys

Public keys prevent symbol mismatch across packages, improve TypeScript declaration quality, and reduce Vue injectable warnings.

```ts
import { inject } from 'vue';
import { CdrFilmstripSymbol } from '@rei/cedar';

const filmstrip = inject(CdrFilmstripSymbol);
```

## Updates

Copy

### Neutral Link

Adds a neutral link variant for default-color display with stacked modifiers.

### Media Object

Updates Media Object to use media-query behavior by default.

## Breaking changes

Copy

### Deprecation of @import

Cedar Sass source is now fully migrated from `@import` to `@use`.

```scss
@use '@rei/cedar/dist/style/cdr-tokens' as cdr;

.my-element {
  background-color: cdr.$cdr-color-primary-600;
}
```

### radius-border

Pulling in cdr-tokens v13 includes fixes for the `radius-border` map value.

## Cedar Tokens v13

Copy

### v13.0.0

Removed deprecated media query mixin tokens:

- `cdr-xs-mq`
- `cdr-sm-mq`
- `cdr-md-mq`
- `cdr-lg-mq`

Use replacements:

1. Media query replacements: `cdr-*-mq-up`
2. Container query replacements: `cdr-*-cq-up`
